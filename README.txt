mywordpress
===========

MyWordpress = Zend Framework &amp; Wordpress Database



Put project into www folder of your Apache webserver (or www folder of your shared host)




Open application/configs/application.ini to set the database connection


resources.db.params.host = <your db host> e.g. localhost 
resources.db.params.username = <your db username> 
resources.db.params.password = <your db pass> 
resources.db.params.dbname = <wordpress database name> 


***********************************************************************

Rename the original Wordpress folder
